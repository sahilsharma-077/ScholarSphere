# ScholarSphere – Student Community Web Platform

## Overview

ScholarSphere is a full-stack student community web platform designed to enable students to create profiles, join communities, share posts, and interact with peers in real time.
The platform focuses on improving collaboration, communication, and knowledge sharing among students within educational institutions.

## Key Features

* User Authentication (Signup / Login)
* Student Profile Management
* Community Feed for Posts and Discussions
* Comment and Like System
* Group / Community Creation
* Real-time Chat (Planned Enhancement)

## Technology Stack

**Frontend:** React.js, HTML5, CSS3, Tailwind CSS
**Backend:** Node.js, Express.js
**Database:** MongoDB Atlas
**Authentication:** JWT (JSON Web Token)
**Version Control:** Git & GitHub

## Project Structure

```
scholarsphere/
 ├── client/        # React frontend
 ├── server/        # Node.js backend
 ├── models/        # Database models
 ├── routes/        # API routes
 ├── controllers/   # Business logic
 └── config/        # Database configuration
```

## Installation

1. Clone the repository

```
git clone https://github.com/yourusername/scholarsphere.git
```

2. Install frontend dependencies

```
cd client
npm install
```

3. Install backend dependencies

```
cd ../server
npm install
```

4. Run the project

```
npm start
```
